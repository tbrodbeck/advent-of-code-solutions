# Advent of Code - Elixir

Advent of Code solutions in Elixir.

## Usage

Run a day's solution:

```bash
mix run -e 'AdventOfCode.solve(1)'
```

Run tests:

```bash
mix test
```

## Structure

```
lib/advent_of_code/day01.ex    # Solution for day 1
test/advent_of_code/day01_test.exs
inputs/day01.txt               # Puzzle input
```
